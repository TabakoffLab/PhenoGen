Analyzing Microarrays

Microarray Analysis Tools allow you to:

  • Analyze precompiled datasets. See "Viewing Datasets"for details.
  • Upload your own data (the raw data files: .CEL files for Affymetrix or .txt files for CodeLink). See "Uploading Your Arrays" for details.
  • Create a dataset from public and private arrays. See "Creating Datasets" for details.
  • View expression values for a list of genes in a dataset. See "Viewing Gene Expression Data" for details.
Take a look

After the microarray data you want is available, you can:

  1. Designate which arrays you want to include in your analysis.
  2. Run the quality control process to ensure the arrays meet basic quality standards.
  3. Group the arrays based on your hypothesis, and normalize the data using one or more methods. Each normalized version can be saved and analyzed independently.
  4. Analyze the normalized versions and save the resulting list of genes.
