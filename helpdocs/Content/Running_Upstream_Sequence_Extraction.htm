Running Upstream Sequence Extraction

The upstream sequence extraction tool is used to extract the upstream genome sequence of a particular gene. 

  1. Choose Genes List Analysis Tools in the main menu.
  2. Click Analyze a gene list. A page displays the gene lists to which you have access.
  3. Click the gene list for which you want to run Upstream Sequence Extraction.
  4. Click the Promoter tab. The Promoter page displays.
  5. Click Run a new Upstream Extraction. The Upstream Extraction page displays.
Take a look

  1. Select the Upstream sequence length.
  2. Click Run Upstream Sequence Extraction. Running Upstream Extraction takes time. When your results are available for viewing, an email is sent to the address you provided in the Registration page.
