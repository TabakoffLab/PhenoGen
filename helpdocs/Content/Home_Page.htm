Website Home Page

The PhenoGen Informatics web site is not only a microarray repository, but also a comprehensive toolbox for analyzing microarray data and researching candidate genes. On the landing page, green options are available to view at any time. Blue options require that you register and log into the website.

Take a look

The Overview option provides descriptions and examples for each of the five major sections:

  • Detailed Transcription Information
  • Downloads
  • Microarray Analysis Tools (login required)
  • Gene Analysis Tools (login required)
  • QTL Tools (login required)

The About option provided links to current datasets, recent publications (documents that relate to the PhenoGen database, gene analysis, and other topics of interest), version information, citations, and links that you may find useful.

The Login/Register option allows you to:

  • Register to use the website.
  • Log in to the website.
  • Retrieve a forgotten password.
