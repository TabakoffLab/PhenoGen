CodeLink Software

CodeLink proprietary software produces quality control flags that show the integrity of each probe on each array. These flags may serve as an indication of inferior arrays. The table lists summary statistics for each array, as well as the number of spots labeled with CodeLink Calls values. For example, arrays having high number of CodeLink Calls other than G (Good), compared to the other arrays within the dataset, can be considered for elimination. See "CodeLink Gene Filtering Procedures" for details about the Number of Probes key (G=Good, L=Near bg signal, C=Contamination, CL=Contamination & Near bg signal, I=Irregular shape, M=Masked, S=Saturated, IS=Irregular shape & Saturated, CI=Contamination & Irregular shape).
