Copying a Gene List

  1. Choose Genes List Analysis Tools in the main menu.
  2. Click Upload or create a new list of genes to use for an analysis. The Create a New Gene List page displays.
Take a look

  1. Enter the Gene List Name.
  2. Select the Organism from the drop-down list.
  3. Enter the Gene List Description.
  4. Make sure Copy Existing Gene List is selected.
  5. Select a gene list from the drop-down list. The gene list populates the box.
  6. Type in new genes and delete existing genes as necessary.
  7. Click Create Gene List. The new gene list is created.
