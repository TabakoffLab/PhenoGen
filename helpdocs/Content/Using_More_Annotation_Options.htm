Using More Annotation Options

More annotation options allow you to select the gene list you want to annotate, then select the links you want iDecoder to search for.

  1. Choose Genes List Analysis Tools in the main menu.
  2. Click Analyze a gene list. A page displays the gene lists to which you have access.
  3. Click the gene list for which you want to perform more annotation.
  4. Click the Annotation tab.
  5. Click More Annotation.
  6. Select one or more target database links and array name(s), if desired.
Take a look

  1. Click Download to download the selected database(s) .
  2. Click Run to view annotation results, including the Gene ID, the database(s) searched, and the links found in each database for your selections.
Take a look

  1. Click any of the different links to open the website for the link you selected.
