Find the genes that are differentially expressed between my disease and control samples

  1. Select a dataset of interest on the Datasets View of the Analyze Microarray Data tab.
  2. Choose one of the normalized versions of the dataset and the differential expression type of analysis. Before you run the analysis, you can filter out some of the probes.
  3. Filter out probes, run the differential expression, and save the resulting gene list.
  4. View the gene list on the Research Genes tab.
